- **TC ID**: Identitas unik test case
- **Nama Test Case**: Deskripsi singkat
- **Input**: Data/action yang diberikan
- **Expected Output**: Hasil yang diharapkan
- **Metode**: Metode pengujian (Black Box, BVA, EP)
- **Status**: PASS/FAIL/PENDING
- **Tanggal Uji**: Kapan diuji
